## LinkedIn

**Agent Browser v0.2.0 — Sessions & Trace Reports**

We shipped logical browser sessions for AI agent workflows. Multi-step deploy validation now leaves a trace: screenshots, console/network signals, and structured JSON/Markdown reports.

**What's new**
- CLI `session` commands and `--session` on browser tools
- MCP `browser_session_*` tools
- Local trace storage under `.agent-browser/`

**Safety unchanged:** private networks blocked, sensitive params redacted, no login automation.

**Honest limitation:** v0.2 traces metadata and evidence — not live page state across separate CLI runs.

Install: `npm install -g @talocode/agent-browser@0.2.0`
