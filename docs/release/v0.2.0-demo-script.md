# Agent Browser v0.2.0 — Demo Script (60s)

**Style:** Talocode builder, dark terminal, honest output, no hype  
**Length:** 45–75 seconds (target ~60s)  
**Renderer:** Terminal screen recording (asciinema, OBS, or similar). No in-repo video tooling.

---

## Scene 1 — Hook (0:00–0:05)

**Caption:** Agents ship code. Who checks the page?

**Voiceover / on-screen text:**
> Your agent deployed a site. Did it actually load?

**Visual:** Dark terminal. Blinking cursor.

---

## Scene 2 — Pain (0:05–0:12)

**Caption:** One-shot checks miss the story.

**Voiceover:**
> Single commands give you a snapshot. Deploy validation needs a trail — screenshots, console, network, and a report.

**Visual:** Quick cut of scattered terminal commands (no fake success banners).

---

## Scene 3 — Introduce Agent Browser (0:12–0:18)

**Caption:** Agent Browser — safe browser automation for AI agents.

**Voiceover:**
> Agent Browser is Talocode's open-source layer for agents to inspect pages, capture evidence, and validate deployments.

**Visual:**
```bash
agent-browser --help
```

---

## Scene 4 — Create session (0:18–0:25)

**Caption:** Start a logical session.

**Command:**
```bash
agent-browser session create --name deploy-check --json
```

**Voiceover:**
> Create a session. Every step gets traced locally.

**Visual:** Copy `sessionId` from JSON output (real output only).

---

## Scene 5 — Smoke check + screenshot (0:25–0:35)

**Caption:** Check the deployed URL.

**Command:**
```bash
agent-browser check https://example.com --session <sessionId> --screenshot-out ./deploy.png --json
```

**Voiceover:**
> Run a deploy-friendly smoke check. Screenshot saved. Status is pass, warn, or fail — no guessing.

---

## Scene 6 — Console + network (0:35–0:45)

**Caption:** Collect signals.

**Commands:**
```bash
agent-browser console --session <sessionId>
agent-browser network --session <sessionId>
```

**Voiceover:**
> Console errors and failed requests show up in the trace. Sensitive query params stay redacted.

---

## Scene 7 — Markdown report (0:45–0:52)

**Caption:** Export the report.

**Command:**
```bash
agent-browser session report <sessionId> --format markdown
```

**Voiceover:**
> One Markdown report: steps, warnings, screenshots, and what to do next.

---

## Scene 8 — MCP + safety (0:52–0:58)

**Caption:** Same workflow via MCP. Safe by default.

**On-screen bullets:**
- `browser_session_create` … `browser_session_report`
- Blocks private networks and unsafe protocols
- No credential automation

---

## Scene 9 — CTA (0:58–1:05)

**Caption:** npm install @talocode/agent-browser@0.2.0

**Voiceover:**
> Logical sessions and trace reports. Install from npm. Link in description.

**Visual:** `npm install -g @talocode/agent-browser` + github.com/talocode/agent-browser

## Recording notes

- Use a dark terminal theme (`#1C1C1C` background, monospace font)
- Run commands for real; do not paste fake JSON
- If Chromium is missing, show the Playwright install message honestly
- v0.2 traces logical sessions — do not claim cookies persist across CLI restarts
